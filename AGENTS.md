# Project Agent Context

## Stack
- React + Vite + Firebase
- Styling: Tailwind CSS
- State: Zustand / Context

## Conventions
- File naming: PascalCase for components, kebab-case for everything else
- Imports: always absolute using @/
- No default exports except for page components

## Hard Limits
- Do not touch package-lock.json or vite.config.js unless asked
- Do not rename exported functions or components (breaks imports)
- Do not install new packages without asking first
